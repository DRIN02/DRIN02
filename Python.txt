1- estrutura de repetição (loop) e uma estrutura de tomada de decisão (if-else):

### Estrutura de Repetição (Loop - for loop):

```python
# Exemplo de um loop que imprime os números de 1 a 5
for i in range(1, 6):
    print(i)
```

Este código irá imprimir os números de 1 a 5.

### Estrutura de Tomada de Decisão (if-else):

```python
# Exemplo de uma estrutura de decisão que verifica se um número é par ou ímpar
numero = 6

if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")
```

Este código verifica se o número é par ou ímpar e imprime a mensagem correspondente.
